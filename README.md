# thepublicwoman

## Overview
The Public Woman is a static blogging website designed to showcase posts in a modern and artsy format. The site features a main feed page that displays blog posts in chronological order and an about page that provides information about the project.

## Features
- Modular upload system for blog posts using XML files.
- Dynamic loading and parsing of posts with JavaScript.
- Modern and artsy design with simple fonts, moving backgrounds, drop shadows, and rounded corners.
- Responsive layout for optimal viewing on various devices.

## Setup Instructions
1. Clone the repository to your local machine.
2. Open the `index.html` file in your web browser to view the main feed.
3. Navigate to the about page using the provided link in the navigation bar.
4. To add new posts, create XML files in the `src/posts/xml` directory and corresponding images in the `src/posts/images` directory.
